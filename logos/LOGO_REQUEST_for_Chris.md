# 🌹 DeadDance logo drive — the easy checklist (for Chris)

Thanks Chris — you're **User #1.** Here's exactly what to gather. Two rules and you're done:

1. **Name each file EXACTLY as shown** (lowercase, underscores, `.png`). The app looks for that exact name.
2. **PNG, transparent background if possible, roughly square, at least 240×240.** (A clean JPG is fine too if that's all that exists — keep the same base name, `.jpg`.)

**Where to get them:** each organization's **own website or press/media kit** (usually a "Press," "Media," "Brand," or footer "Logo" link). That's the official mark, best quality.

**Drop finished files here:** the `logos/` folder (Issa will point you to it / a shared drive). Miss one? No problem — the app just shows an icon until the file lands.

## Keeping it clean (please read)
- Use only each org's **official logo from their own site or press kit** (or one the org hands you directly). Don't pull from random image searches or re-color/redraw anything.
- We're **offering each of these a free feature** — a quick "we're putting you on the DeadDance map, cool if we use your logo?" is the friendly, clean way, especially for the schools, police, and fire. When in doubt, ask them; the logo *is* their yes.
- DeadDance isn't claiming any endorsement, and any org can have their logo removed on request, no questions asked.
