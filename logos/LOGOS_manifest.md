# DeadDance ad logos — drop files here

Each rolling ad looks for its logo in this folder. **Until a file exists, the emoji shows** (nothing breaks). Drop a PNG (transparent background best, ~square, ≥240px) with the EXACT filename below and it appears on the next roll.

Format: PNG (or swap the reference to .svg in index.html if you have vector). No file = graceful emoji fallback.

## Musikfest bands (458 unique — optional)
Each band ad looks for `logos/band_<slug>.png`, where `<slug>` = the band name lowercased, `&`→`and`, non-alphanumerics→`_`. Example: "Grupo Niche" → `logos/band_grupo_niche.png`. Only the band currently on screen fetches its logo, so missing files cost nothing. Fill in the ones you want; the rest ride the emoji.
